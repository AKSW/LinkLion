LinkLion Ontology Vocabulary

Algorithm

An algorithm used in the link discovery task, i.e. discovering new links between two datasets. An algorithm is connected to a :LDFrameworkVersion within a prov:wasAssociatedWith property.
URIhttp://www.linklion.org/ontology#Algorithm
typeClass
namespacehttp://www.linklion.org/algorithm/
subclass ofhttp://www.w3.org/ns/prov#Activity

Link

A link discovered during the link discovery process. A link is connected to a :Mapping using a prov:wasDerivedFrom property.
URIhttp://www.linklion.org/ontology#Link
typeClass
namespacehttp://www.linklion.org/link/
subclass ofhttp://www.w3.org/ns/prov#Entity

Link Discovery Framework

A link discovery framework. Each framework has one or more doap:releases, which belong to the :LDFrameworkVersion class.
URIhttp://www.linklion.org/ontology#LDFramework
typeClass
namespacehttp://www.linklion.org/framework/
subclass ofhttp://www.w3.org/ns/prov#Agent
subclass ofhttp://usefulinc.com/ns/doap#Project

Link Discovery Framework Version

A release of a link discovery framework. A framework version has a doap:revision datatype property.
URIhttp://www.linklion.org/ontology#LDFrameworkVersion
typeClass
namespacehttp://www.linklion.org/version/
subclass ofhttp://www.w3.org/ns/prov#Agent
subclass ofhttp://usefulinc.com/ns/doap#Version

Mapping

A mapping between two datasets. A mapping is connected to an :Algorithm within a prov:wasGeneratedBy property. Each mapping has a prov:generatedAtTime datatype property.
URIhttp://www.linklion.org/ontology#Mapping
typeClass
namespacehttp://www.linklion.org/mapping/
subclass ofhttp://www.w3.org/ns/prov#Entity
